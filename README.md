# Hi, I'm Taylor Go

## What Is This?

Over the years, I've had a [personal blog and CMS written in Rails](https://github.com/hiimtaylorjones/Hiimtaylorjones) that I used to learn Ruby but also expirment with new technologies within.

For a long time, the Rails app suited what I was interested in - writing Ruby and exploring all the Javascript frameworks that have come and gone over the years. 

In my work, I started to get the opportunity to write more Go but never really took the time to explore duplicating writing something like a blog CMS in Go. This project is the result of me finally sitting down to make time for exploration!

I used Claude Code to help me scaffold and develop things step by step. What I didn't do here was tell Claude: "Hey, make me a copy of my Rails-based blog in Go. Then commit and push." 

The initial experience in developing this was more along the lines of asking Claude to develop a guided plan for me to follow in creating this app.
